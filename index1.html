Frequently Asked Questions

Find answers to common questions about the Frontend Battle competition.

Submissions are evaluated based on four main criteria: Design Accuracy (40%), Responsiveness (25%), Innovation/Enhancements (20%), and Code Quality & Best Practices (15%). Each submission is reviewed by a panel of expert judges.

You must implement at least 6 features from the provided list to have a valid submission. However, implementing more features with high quality can improve your score in the Innovation/Enhancements category.

Yes, implementing both light and dark modes is one of the mandatory features for a valid submission. The toggle should be functional and the themes should be well-designed for consistency.

While you can use frameworks, the competition focuses on core frontend skills. We recommend using plain HTML, CSS, and JavaScript to demonstrate your fundamental knowledge. If you do use a framework, you'll be judged on the same criteria.
